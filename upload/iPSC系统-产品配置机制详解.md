## 1. 配置化架构概述

### 1.1 三层架构设计

```
┌─────────────────────────────────────────────────────────────────┐
│  第1层：系统代码（开发一次，无需改动）                            │
│  · 基础数据模型（批次、任务、质检、CoA）                          │
│  · 流程引擎（状态流转、任务调度）                                 │
│  · 表单渲染引擎（根据配置动态渲染）                               │
│  · 通用业务逻辑（权限、通知、追溯）                               │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  第2层：产品配置（用户/管理员配置）                               │
│  · 产品定义（名称、编码、规格、有效期）                           │
│  · 流程模板（生产环节、执行顺序）                                 │
│  · 表单模板（每个环节的字段）                                     │
│  · 质检模板（检测项目、方法、标准）                               │
│  · CoA模板（显示字段、附件）                                      │
│  · 配方/BOM（试剂盒原料组成）                                     │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  第3层：业务数据（日常操作产生）                                  │
│  · 批次记录、生产任务、质检结果、CoA文档                          │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 核心价值

| 场景 | 传统开发 | 配置化方式 | 效率提升 |
|------|----------|------------|----------|
| 新增iPSC新品系 | 2周 | 10分钟 | 99% |
| 新增分化细胞类型 | 2周 | 10分钟 | 99% |
| 新增试剂盒配方 | 1周 | 30分钟 | 98% |

## 3. 新增分化细胞产品示例（对比）

### 分化细胞产品与试剂盒的配置差异

| 配置项 | 试剂盒 | 分化细胞产品 | 差异说明 |
|--------|--------|--------------|----------|
| **产品类型** | 试剂盒 | 细胞产品 | 系统预定义 |
| **生产流程** | 配制→分装→质检 | 种子准备→复苏→扩增/分化→收获→质检 | 环节不同 |
| **表单重点** | 配方展开、多组分 | 细胞培养记录、形态观察 | 字段不同 |
| **配方/BOM** | ✅ 需要 | ❌ 不需要 | 试剂盒特有 |
| **质检重点** | 无菌、pH、效力 | 活率、形态、支原体、标志物 | 项目不同 |

## 5. 总结

### 5.1 配置化程度

| 配置项 | 配置方式 | 技术难度 | 配置者 |
|--------|----------|----------|--------|
| 产品基本信息 | 表单填写 | ⭐ | 管理员 |
| 生产流程 | 拖拽+选择 | ⭐⭐ | 生产主管 |
| 表单模板 | 可视化设计器 | ⭐⭐ | 生产主管 |
| 配方/BOM | 表格填写 | ⭐⭐ | 研发/生产主管 |
| 质检模板 | 表单填写 | ⭐ | QA经理 |
| CoA模板 | 勾选配置 | ⭐ | QA经理 |

### 5.2 新增产品时间估算

| 产品类型 | 基于复制 | 从头配置 | 主要工作 |
|----------|----------|----------|----------|
| iPSC新品系 | 10分钟 | 30分钟 | 修改产品名称、编码 |
| 分化细胞新品 | 15分钟 | 1小时 | 调整分化方案、质检项目 |
| 试剂盒新品 | 30分钟 | 2小时 | 配置配方/BOM |

### 5.3 关键设计原则

1. **80%预置 + 20%自定义**：系统预置常用模板，用户只需微调
2. **可视化配置**：拖拽、勾选，无需编写代码
3. **版本控制**：支持配置版本管理，可追溯、可回滚
4. **权限隔离**：不同角色只能修改自己负责的模块
5. **即时生效**：配置保存后立即生效，无需重启系统
