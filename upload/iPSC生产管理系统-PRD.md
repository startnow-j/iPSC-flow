## 1. 产品概述

### 1.1 产品定位

**iPSC生产管理系统**是连接服务项目、细胞产品、试剂盒三大业务线的**统一生产管理平台**，定位为现有细胞库存管理系统和ELN系统的补充，形成完整的实验室数字化管理闭环。

### 1.2 产品愿景

> 让每一个批次都可追溯，让每一份CoA都可信赖，让生产管理从混乱走向有序。

### 1.3 核心价值主张

| 价值点 | 具体描述 | 业务收益 |
|--------|----------|----------|
| **记录数字化** | 替代文件夹管理，所有生产记录在线完成 | 杜绝丢失和缺漏，人员流动不再导致数据丢失 |
| **流程驱动** | 系统引导用户按正确顺序完成每个步骤 | 降低新人培训成本，减少操作遗漏 |
| **数据联动** | 生产数据自动流转到质检、CoA、库存 | 减少重复录入，提升效率 |
| **一键追溯** | 任意批次全生命周期信息一键查询 | 满足质量审计要求，快速响应客户查询 |
| **CoA自动化** | 从质检数据自动生成CoA草稿 | 缩短交付周期，减少人工错误 |

### 2.3 用户旅程地图

#### 场景1：细胞产品组操作员 - 新建生产批次

```
登录系统 → 进入"批次管理" → 点击"新建批次" 
    ↓
选择产品类型（iPSC细胞）→ 系统自动生成批号
    ↓
选择种子来源（扫描/选择种子批号）→ 系统自动带出种子信息
    ↓
确认生产计划 → 提交创建 → 系统生成eBPR任务
    ↓
进入"我的任务" → 开始按步骤录入生产记录
    ↓
完成生产 → 提交质检 → 等待质检结果
```

#### 场景2：QA人员 - 审核CoA

```
登录系统 → 进入"CoA管理" → 查看"待审核"列表
    ↓
点击待审核CoA → 查看完整批次信息+质检数据+附件
    ↓
核对数据一致性 → 审核通过/退回修改
    ↓
如通过：CoA状态变为"已批准"，批次可放行
如退回：填写退回原因，通知相关人员修改
```

#### 3.3.3 功能详情

**3.3.3.1 批号生成规则**

**细胞产品批号格式：**
```
[产品代码]-[年月日]-[流水号]-[代次]

示例：IPSC-260410-003-P5

字段说明：
- 产品代码：IPSC(诱导多能干细胞), NPC(神经前体细胞), CM(心肌细胞)等
- 年月日：YYMMDD格式，如260410表示2026年4月10日
- 流水号：当日该产品的第N批，3位数字，如001, 002
- 代次：细胞代次，如P5表示第5代
```

**试剂盒批号格式：**
```
[试剂盒代码]-[年月]-[流水号]

示例：NPC-KIT-2604-001

字段说明：
- 试剂盒代码：NPC-KIT(神经分化试剂盒), CM-KIT(心肌分化试剂盒)等
- 年月：YYMM格式，如2604表示2026年4月
- 流水号：当月该试剂盒的第N批，3位数字
```

**3.5.3.3 自动判定规则**

```javascript
// 判定逻辑示例
function autoJudge(result, standard) {
  if (result.type === 'number') {
    // 数值型：与阈值比较
    return result.value >= standard.threshold ? 'PASS' : 'FAIL';
  } else if (result.type === 'select') {
    // 选择型：与期望值匹配
    return result.value === standard.expected ? 'PASS' : 'FAIL';
  }
}

// 综合判定
function finalJudge(itemResults) {
  const allPass = itemResults.every(r => r.judgment === 'PASS');
  return allPass ? '合格' : '不合格';
}
```

### 8.1 批号生成算法

```python
def generate_batch_no(product_code, passage=None):
    """
    生成细胞产品批号
    格式: [产品代码]-[年月日]-[流水号]-[代次]
    示例: IPSC-260410-003-P5
    """
    from datetime import datetime
    
    today = datetime.now()
    date_str = today.strftime('%y%m%d')  # 如: 260410
    
    # 查询当天该产品的最大流水号
    max_seq = get_max_sequence(product_code, date_str)
    new_seq = max_seq + 1
    
    batch_no = f"{product_code}-{date_str}-{new_seq:03d}"
    if passage:
        batch_no += f"-P{passage}"
    
    return batch_no
```
